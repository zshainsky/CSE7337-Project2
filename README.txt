README
A readme file with complete description of used software, installation, compilation and execution instructions

Software Used:
1) Scrapy
2) Python


Install Instructions:
Scrapy:
Please refere to the Scrapy online documentation (http://doc.scrapy.org/en/0.24/intro/install.html#intro-install)

Python:
Please refere to the Python online documentation (https://www.python.org/downloads/)


Execution Instructions:
Scrapy:
Enter commands in commandline tool after downloading Scrapy
1) From the command line/terminal, navigate into the /zCrawler folder

2) Run Spider from the terminal with the following command
	>> scrapy crawl <spider name>

All above instructions are found at http://doc.scrapy.org/en/0.24/intro/tutorial.html. Please references this for the page to set up a project.

Results:
Results will appear in  the zCrawler folder in a folder called CrawlerResults
